Konfigurasi jaringan memungkinkan akses ke komputer cadangan untuk keperluan spesifik. Dalam skenario ini, komputer 1 berfungsi sebagai komputer utama untuk aktivitas online, sementara komputer 2 bertindak sebagai cadangan yang menjalankan aplikasi Kiwix Serve, sebuah server Wikipedia offline berbasis web. Untuk mengakses Kiwix Serve dari komputer 1, kedua komputer perlu dihubungkan. 

Meskipun bisa dilakukan melalui WiFi, rekomendasi adalah menggunakan koneksi ethernet untuk kinerja transfer data yang lebih cepat dan fleksibilitas.  Dengan ethernet, pengaturan IP pada kedua komputer harus berada pada jaringan yang sama.  Ini memungkinkan kontrol atas akses internet pada komputer 2,  memberikan opsi untuk mematikan akses internet jika tidak diperlukan.  Pendekatan ini menyediakan cara yang relatif sederhana untuk menghubungkan dan memanfaatkan komputer cadangan.
